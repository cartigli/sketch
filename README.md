SKETCH is designed to allow the user to draw and calibrate lines on a given image.

After the initial (simple) calibration of a single line's distance, the user is allowed to continue drawing lines on the image, and each one will be defined at its given length 
determined by the initial calibration. img_calli.py runs off of a single image. 

The next script will take the input of a (live) video feed to let the user calibrate and then move objects arouund for more dynamic measurements.
